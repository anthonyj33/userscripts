# To install and use userscripts, I recommend using [Tampermonkey](https://www.tampermonkey.net)

- [Open Canvas calendar event in new tab](./open_canvas_event_in_new_tab.user.js)
  - Enables Ctrl/Cmd+Click or middle click on a Canvas calendar event to open the corresponding page in a new tab.
  - Keeps the calendar page focused.
  - Some configuration may be required for other educational institutions -- see the options in the metadata block.
